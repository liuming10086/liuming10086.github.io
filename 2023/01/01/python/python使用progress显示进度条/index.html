使用

在progress库中，只用到它的「三个实例方法」，就可实现一个进度条的动作：

「start」 进度条初始化，给它一个目标上限值，例如到 100% 结束，max_value 参数给个 100 就 OK 了。
「update」 更新进度条进度，这个方法执行一次，进度条就会更新一次，value 参数就是要更新的值。
「finish」 关闭进度条，不想再对进度条进行操作，执行它。
